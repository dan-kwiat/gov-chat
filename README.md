# Gov Chat

Prototype of a gov.uk page with a built-in chatbot.

## Installing

```bash
cp .env .env.local # then fill in the missing values in .env.local
pnpm install
```

## Developing

```
pnpm run dev
```

## Building

```
pnpm run build
```
